# 目录解释
***app...........项目应用目录***  
    Controller....控制器目录
    Extend........扩展目录  
    Middleware....中间件  
    Model.........模型目录  
    Validate......验证器目录  
    View..........视图目录  
***cache.............配置缓存目录***  
***config............应用配置目录***    
    aop...........切片控制目录  
    app...........项目配置目录  
    cache.........缓存配置文件  
    database......数据库配置文件   
    redis.........redis配置文件  
***libs..............框架核心目录***
